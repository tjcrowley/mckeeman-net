# mckeeman.net — one-page novels site

## Files
- `index.html` — static one-page ad/landing page for Darren Mckeeman novels.

## Deploy options

### Option A: Any static host (fastest)
Upload `index.html` as site root.

### Option B: Nginx/Apache VPS
Copy `index.html` to the document root for `mckeeman.net`.

### Option C: Cloudflare Pages / Netlify / Vercel
Create a static project and publish this folder.

## Content pulled from
- https://www.amazon.com/stores/Darren-Mckeeman/author/B091NC8PD3

Update book links/titles directly in `index.html`.
